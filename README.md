test dpc. owndroid. offline mdm. dhizuku. תומכות בהעברת הבעלות מאחת לשניה.

הנה הקישור להורדה: https://github.com/IsraelBer/SecureGuardMDM/releases/download/0.4.5/Abloq-release.apk

הדבר יעיל לצורך:

1.  אפשר להגדיר הגדרות שקיימות רק באפליקציות האחרות, לדוגמה ב test dpc, ואחר כך כשתעבירו בעלות, ההגדרות הללו ישארו!

2.  את האפליקציה test dpc ניתן להתקין אחרי איפוס ללא צורך במצלמה או בחיבור למחשב, פשוט בוחרים במסך הפתיחה להתחבר לחשבון בגוגל, ובמקום כתובת אימייל, מקלידים afw#testdpc ומיד האפליקציה מותקנת ומוגדרת כמנהל מכשיר, ואז נכנסים, מחפשים transfer, לוחצים, בוחרים בגרסת האיי בלוק המצורפת, (אחרי שהתקנתם כמובן)
